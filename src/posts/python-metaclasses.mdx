The simplest metaclass is `type`. When `type` receives one parameter as input, it returns the type of the object passed as a parameter. In this case, it does not work as a metaclass. When `type` receives three parameters, it acts as a metaclass and creates a class based on the passed parameters.


```python
>>> MyClass = type('MyClass', (object,), {'my_attribute': 0})
>>> type(MyClass)
<class 'type'>
>>> o = MyClass()
>>> o.my_attribute
0
```
